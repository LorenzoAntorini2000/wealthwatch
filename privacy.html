Privacy Policy

WealthWatch is a personal finance application built for private use by a single individual. It is not a commercial product and has no user base beyond its owner.

1. Who We Are

WealthWatch is a personal wealth tracking application operated by Lorenzo Antorini for private use only. It is not operated as a business and does not offer services to the public.

2. What Data Is Collected

WealthWatch stores only the data explicitly entered or authorised by its sole user:

  • Account names, types, and balances entered manually
  • Historical balance snapshots taken by the user
  • Bank account connection tokens obtained through the Open Banking (PSD2) consent flow
  • Cryptocurrency portfolio valuations retrieved via API

No data is collected about behaviour, location, device, or browsing activity.

3. How Data Is Used

All data is used exclusively to display a personal net worth dashboard to the application owner. It is not processed for any other purpose, not sold, not shared, and not used for advertising or profiling.

4. Data Storage

Data is stored in a private Supabase database protected by authentication. Access is restricted to the sole owner of the application via email and password login. Row-level security policies ensure no data is accessible to other parties.

5. Third-Party Services

WealthWatch integrates with the following external services on behalf of its owner:

  • Enable Banking — provides read-only access to bank account balances via PSD2 Open Banking APIs. No credentials are stored by WealthWatch; authentication is handled directly between the user and their bank.
  • Crypto.com — provides read-only access to cryptocurrency portfolio balances via API.
  • Supabase — provides database and authentication infrastructure.

Each of these services operates under its own privacy policy and data protection terms.

6. Data Sharing

No personal or financial data is shared with any third party beyond the integrations listed above, which are strictly necessary for the application to function. There is no advertising, analytics, or data monetisation of any kind.

7. Your Rights

As this application is operated solely for personal use by a single individual, standard data subject rights apply trivially — the sole user is also the sole data controller and can delete all data at any time directly from the application or by contacting Supabase.

8. Contact

For any questions regarding this privacy policy, please contact: lorenzo@antorini.net
