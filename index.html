Enable Banking

Enable Banking is a third-party Open Banking aggregator that gives WealthWatch read-only access to your bank account balances via the PSD2 API — without ever storing your banking credentials.

  1. Go to enablebanking.com and create a free account.
  2. From the dashboard, navigate to API Applications → + New Application.
  3. Fill in all mandatory fields:
    • Name: anything, e.g. "WealthWatch"
    • Description: e.g. "Personal wealth tracking app"
    • Data protection email: your email address
    • Privacy policy URL: https://lorenzoantorini2000.github.io/wealthwatch/privacy.html
    • Terms of service URL: https://lorenzoantorini2000.github.io/wealthwatch/terms.html
    • Redirect URLs: https://lorenzoantorini2000.github.io/wealthwatch/
    • Set type to Production
  4. Once the application is created, a .pem private key file will be downloaded — save it securely, as it cannot be retrieved again.
  5. Copy the Application ID (a UUID like xxxxxxxx-xxxx-...) and paste it into the App ID field below.
  6. Open the .pem file in a text editor, copy the full contents (including the -----BEGIN/END----- lines) and paste them into the Private Key field below.
  7. Important: before linking a bank account in WealthWatch, open your production application on Enable Banking, click Link accounts, and complete at least one bank connection there first — this verifies your identity under the bank's consent framework. When linking in WealthWatch, use the exact bank name as shown on Enable Banking.

Generate an RSA key pair, register the public key in your Enable Banking developer dashboard, and paste the private key here.

Crypto.com

Crypto.com Exchange provides a read-only API to retrieve your portfolio's total market value in real time. This is the exchange product (exchange.crypto.com), not the main app.

  1. Go to exchange.crypto.com and sign in.
  2. Click your profile icon (top right) → Manage Account.
  3. Navigate to API Management → + Create New API Key.
  4. Set a label (e.g. "WealthWatch"), leave permissions as Can Read only — do not enable trading or withdrawal permissions.
  5. Complete 2FA verification.
  6. Copy the API Key and Secret Key immediately — the secret is shown only once and cannot be retrieved again.
  7. Paste both values into the fields here.

Create a read-only API key in the Crypto.com Exchange app under Settings → API Keys.

Interactive Brokers

Interactive Brokers' Flex Web Service lets you export account data — including total Net Asset Value — via a scheduled report. WealthWatch uses this to read your portfolio's EUR balance.

  1. Log in at interactivebrokers.com.
  2. Type "Flex Queries" in the search bar and open the page.
  3. First, enable the Flex Web Service: click Flex Web Service Configuration (top right of the page) → click Generate Token → copy the token immediately and paste it into the Flex Token field here.
  4. Back on the Flex Queries page, click + Create next to Activity Flex Query.
  5. Name it (e.g. "WealthWatch NAV"), then under Sections find Net Asset Value (NAV) in Base Currency and select the following fields: Account ID, Cash, Stock, Total.
  6. Set Date Period to Last Business Day and format to XML, then save.
  7. Click the ℹ icon next to your saved query to find the Query ID — copy it and paste into the Query ID field here.

Create a Flex Query in IBKR Account Management → Reports → Flex Queries, then generate a Flex Web Service token.
